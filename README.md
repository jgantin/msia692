## ✏️ How to Edit Content (No coding required)

### Editing text on GitHub.com:
1. Go to the repo on GitHub
2. Click on `index.html`
3. Click the **pencil icon** (Edit this file) in the top right
4. Make your changes
5. Scroll down → click **"Commit changes"** → **"Commit directly to main"**
6. The live site updates automatically in ~1 minute

### Things you can easily change:
- Mass schedule times and location
- Contact email address
- Activity descriptions
- Any text in the page

---

## 🖼️ Adding Photos

1. Go to `assets/images/` in the repo
2. Click **"Add file" → "Upload files"**
3. Drag and drop your photo
4. Commit the upload
5. Reference it in `index.html` like: `<img src="assets/images/your-photo.jpg" />`
